# gerber2stl
Converts Gerber files to STL files

# How to compile and run
run ```make``` to compile
<br/>
then ```./gerber2stl <directory with gerber files> <output directory>```

# Example
```make``` <br/>
```./gerber2stl ./gerber_files/ ./output_dir/```
